## 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

## 입력
첫째 줄에 정수 N(0 ≤ N ≤ 12)이 주어진다.

## 출력
첫째 줄에 N!을 출력한다.


## 입출력 예
### 예제 입력1
```
10
```
### 예제 출력1
```
3628800
```
### 예제 입력2
```
0
```
### 예제 출력2
```
1
```

## 문제 접근
1. 재귀를 사용해 문제를 해결
2. N을 1씩 줄여가며 곱하는 재귀함수를 선언
3. N=0 일때 N!==1 이므로 base case를 맞춰준다.

--- 

## 제출 답안

```python
N = int(input())

def func(N):
    if N < 1:
        return 1
    return N * func(N-1)

print(func(N))
```
