### [2754](https://www.acmicpc.net/problem/2754)
<details>
<summary>풀이</summary>

```python
grade = input()
rating = 0.0
try:
    match grade[1]:
        case "+":
            s = 0.3
        case "-":
            s = -0.3
        case _:
            s = 0
    match grade[0]:
        case 'A':
            rating = 4.0 + s
        case 'B':
            rating = 3.0 + s
        case 'C':
            rating = 2.0 + s
        case 'D':
            rating = 1.0 + s
    print(rating)
except IndexError:
    print(rating)
```
